# Documentación del Sistema

Este directorio concentra la documentación operativa y técnica del proyecto.

## Documentos principales

- `docs/SYSTEM_FULL_DOCUMENTATION.md`: guía integral del sistema (arquitectura, módulos, datos, seguridad, operación, scripts y QA).
- `docs/DEPLOYMENT_SECURITY.md`: hardening y checklist de seguridad para producción.
- `docs/CANARY_ROLLOUT_RUNBOOK.md`: despliegue gradual 10% -> 50% -> 100% con criterios de avance y rollback.
- `docs/MARKETING_WEBSITE_PLAN.md`: estrategia web comercial (landing, embudo, tracking, KPIs y plan de lanzamiento).

## Documentos de soporte en raíz

- `README.md`: overview rápido del proyecto y setup base.
- `plan.md`: roadmap histórico/evolutivo del producto (no reemplaza runbooks operativos).

## Ruta recomendada de lectura

1. `README.md`
2. `docs/SYSTEM_FULL_DOCUMENTATION.md`
3. `docs/DEPLOYMENT_SECURITY.md`
4. `docs/CANARY_ROLLOUT_RUNBOOK.md`
